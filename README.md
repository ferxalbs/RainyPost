# RainyPost

A native macOS API client built with SwiftUI. Fast, offline-first, and designed for developers.

![macOS](https://img.shields.io/badge/macOS-14.0+-blue)
![Swift](https://img.shields.io/badge/Swift-5.9+-orange)
![License](https://img.shields.io/badge/License-Custom-green)

## Features

- **Native macOS Experience** - Built with SwiftUI, feels at home on your Mac
- **Offline-First** - All data stored locally, no cloud required
- **Environment Variables** - Manage different environments seamlessly
- **Request Collections** - Organize your API requests
- **Import/Export** - Compatible with Postman/Insomnia collections
- **cURL Support** - Copy requests as cURL commands

## Installation

### Requirements
- macOS 14.0 (Sonoma) or later
- Xcode 15.0 or later (for building from source)

### Build from Source
```bash
git clone https://github.com/YOUR_USERNAME/RainyPost.git
cd RainyPost
open RainyPost.xcodeproj
```

Build and run in Xcode (⌘R).

## Usage

1. **Create a Workspace** - Start by creating a new workspace for your project
2. **Add Requests** - Create GET, POST, PUT, PATCH, DELETE requests
3. **Set Environments** - Configure variables for different environments
4. **Send & Inspect** - Execute requests and view responses
